How MultiMC users will update to Minecraft 1.17

The new Minecraft release requires Java 16. Unlike previous versions, Java 16 from Oracle is not a free download — it has to be obtained from vendors like Microsoft or AdoptOpenJDK. Thankfully, the process is relatively straightforward!

Visit the website for either Microsoft OpenJDK or AdoptOpenJDK. Both of these products are identical and will run on Windows, macOS, and Linux.

Make sure to choose the x64 .msi file for Java 16. Other versions will not work.

Download this file and run it.

Download this file and run it. Click next.

Click "Next".

Wait for it to finish installing.

Launch MultiMC, and click either Edit Instance or Settings.

Important: Use Settings only if you wish to switch all your instances over to Java 16. This will break any pre-1.17 Forge instances!

Click Settings, go under the Java tab, check ✅ Java Installation, then click Browse....

Locate javaw.exe. If you installed Microsoft Java for Windows, this will be at a path that looks like C:\Program Files\Microsoft\jdk-16.0.1.9-hotspot\bin\javaw.exe, but may not be exactly this.

Run the sanity check. It should say Java version reported: 16. If it says an older version, you skipped a step and the game will not launch.

Under the Version tab, update your Minecraft to 1.17, update your fabric, and update your mods.
